Contribute to Studio Keys

Do you have .studio design files? Do you want to help me make this program better?

Studio Keys was created to provide artists and graphic designers who used InVision with a way to regain access to their designs. While Studio Keys already handles a lot of design elements quite well, there are still many areas for improvement to make it more precise and to improve the quality of the mockups rendered. For example, you may have noticed instances of text elements overlapping on each other in one of your SVGs. That is something I am in the process of figuring out.

The best (and arguably only) way that I've been able to create or make the program is by using a supply of .studio files to test the features and learn about how those files are structured and how to disassemble and re-assemble them. The more diverse a portfolio of .studio file samples I have, the more I can continue to refine this program. Alternatively, if you'd just like to provide a few sentences of feedback on the application and your experience (positive or negative), it would still mean the world!
